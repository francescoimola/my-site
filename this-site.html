About this site

Here is where I gather my work, hobbies, and passions. This site is a a portfolio of past and work-in-progress projects and a greenhouse where new ideas grow. It expands my digital garden that exists across Are.na and Twitter.

I designed this site by hand using only HTML, CSS, and 5 lines of JavaScript. I built it with accessibility at the forefront, and adopting sustainable web design practices. The open-source typeface in use is Arimo by Steve Matteson.

Do you like it and want to make something similar? The source code is available on GitHub and you can drop me a message if you have any questions.
