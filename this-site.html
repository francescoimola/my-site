About this site

This site is my front garden. My safe space on the web. I've tried lots of options in the past. Even the flashy ones with top interactions. Nothing stuck more than being able to do this myself, by hand.

I have a slow, uneven online presence, with the more obscure but also more exciting bits up on Are.na. You can consider this site part of that digital garden.

Pages are built using just HTML and CSS. 0 JavaScript. I set things up with accessibility at the forefront, and adopting sustainable web design practices. The open-source typeface in use is Arimo by Steve Matteson.

Do you like it and want to make something similar? The source code is available on GitHub and you can drop me a message if you have any questions.
